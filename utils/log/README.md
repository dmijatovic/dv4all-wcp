# @dv4all/log

**`THIS LIBRARY IS USED BY OTHER WEB COMPONENT MODULES.`**

Console log utility function. Later it might be extended with other logging options?

## Usage

```javascript
import {consoleLog} from '@dv4all/log'

consoleLog({
  group: null, //null - no group
  msg:"log message"
  env:"test" //default
})

```
